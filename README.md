# 🔴🔵 Visor Estereoscópico Anaglifo

## 📌 Descripción

Este proyecto consiste en el desarrollo de una aplicación web responsiva que permite visualizar imágenes estereoscópicas utilizando dos métodos:

* 👀 **Side by Side (SBS)**: Visualización de imágenes lado a lado.
* 🔴🔵 **Anaglifo**: Visualización con efecto 3D mediante lentes rojo/cian.

La aplicación permite alternar entre ambos modos de visualización mediante botones interactivos, ofreciendo una experiencia visual dinámica y educativa.

---

## 🎯 Objetivo

Desarrollar una aplicación web que permita la visualización de imágenes estereoscópicas, integrando tecnologías modernas como Bootstrap 5 y JavaScript, cumpliendo con los requisitos de diseño responsivo e interacción dinámica.

---

## 🧠 Funcionamiento

Cada imagen cuenta con dos versiones:

* **SBS (Side by Side)** → Imagen con vista izquierda y derecha en una sola imagen
* **Anaglifo** → Imagen procesada con filtros rojo/cian

Mediante JavaScript, se cambia dinámicamente la fuente (`src`) de las imágenes al presionar los botones:

* Botón **Side by Side** → muestra imágenes SBS
* Botón **Modo Anaglifo** → muestra imágenes anaglifo
